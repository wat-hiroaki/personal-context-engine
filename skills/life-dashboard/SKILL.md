# Life Dashboard

全ライフデータを横断的に分析し、インサイトを提供する統合スキル。

> **Status**: v0.3 で実装予定。現在はプレースホルダー。

## Trigger Phrases

- 「今月の支出レポートを見せて」
- 「健康関連の出費はいくら？」
- 「来週買うべきものリスト」
- 「今週のライフレポートを作って」

## Data Sources

| データ | ソース | 統合方法 |
|--------|--------|---------|
| 所有物・購入履歴 | PCE (SQLite) | 直接クエリ |
| 健康データ | Google Fit / Apple Health | 既存スキル or エクスポート |
| 食事ログ | OpenClawスキル | 既存データを参照 |
| 筋トレログ | ユーザー記録 | 既存データを参照 |
| 習慣チェックリスト | ユーザー記録 | 既存データを参照 |

## Cross-Domain Analysis Examples

- 「トレーニング強度上げてるけどプロテイン在庫大丈夫？」→ 筋トレログ + 消耗品DB
- 「今月の健康関連支出はいくら？」→ 購入履歴 + カテゴリフィルタ
- 「習慣の達成率と睡眠の相関」→ 習慣ログ + Google Fit
- 「来週買うべきものリスト」→ 消耗品予測 + 過去の購入パターン

## Heartbeat Jobs

| ジョブ名 | スケジュール | 内容 |
|----------|-------------|------|
| 消耗品チェック | 毎日 9:00 | 消耗品の残量を確認し、なくなりそうなアイテムを通知 |
| 週次ライフレポート | 毎週日 20:00 | 習慣・栄養・トレーニング・支出のサマリーを生成 |
| 購入提案 | 毎週土 10:00 | 消耗パターンと在庫から購入推奨リストを作成 |
